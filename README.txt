This contains several files. The titanic.csv is the datafile, which is used for the task. 
However, the user can specify the name of the file that they want to use.
In the command line, write the following:
python3 main.py <filepath> <choice>
filepath represents the path of the data file(titanic.csv in this case)
choice represents the part of the code that will be executed. Different numbers represent that.
1 - print the first 20 rows of the dataframe
2 - print and visualize the correlation matrix
3 - train the neural network
Some functions and classes are imported from the neural_network.py file
